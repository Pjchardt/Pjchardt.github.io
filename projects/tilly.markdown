  <h3 align="left">Tilly</h3>
  <p>
    Tilly is a performative machine that generates haiku based on what it observes in the world around it. Details of certain performances, documentation, and a full list of Tilly's performances follow. The latest haiku generated by Tilly can be retrieved by calling the <a style="display:inline; color:#0062cc;" href="http://142.93.240.55/getHaiku">API endpoint</a>.  [note: Tilly spends most of its time offline]
  </p>

  <h4 align="left">Tilly and Telly</h4>
  <p>
    Performance lasting seven days, Tilly sat in an empty room watching network television 24 hours a day on an old cathode ray television. Every 5 minutes Tilly would speak a haiku to the empty room then send the haiku to the Tilly server. An API allowed anyone to request the latest haiku from the server.
  </p>
